# Hearth & Ink - Project Roadmap

## 🎯 The Vision
"Hearth & Ink" turns house photos into minimalist, hand-drawn architectural pen-and-ink sketches. A high-value, "closing gift" for realtors and homeowners.

## 🛠 Tech Stack
- **Frontend:** Next.js (hosted on Vercel)
- **AI Engine:** Gemini 3 Pro Image (via `nano-banana-pro` skill)
- **Payments:** Stripe (Checkout for single orders)
- **Database:** Supabase (for order tracking and image storage)
